# AdVision AI — Sitio Web Oficial

Sitio web de AdVision AI: academia digital de cursos prácticos de Inteligencia Artificial para emprendedores latinoamericanos.

## Tecnología
- HTML5 + CSS3 + JavaScript vanilla
- Sin dependencias externas (solo Google Fonts via CDN)
- Deploy automático en Vercel

## Despliegue
Este sitio se despliega automáticamente en Vercel al hacer push a `main`.
